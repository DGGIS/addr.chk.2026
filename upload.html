1. 엑셀 파일 업로드

주간/월간 보고 템플릿 파일을 업로드하세요

「기초번호판_주간월간보고_템플릿.xlsx」 양식의 파일을 올리면
기준일자를 자동 인식해 해당 주차 데이터로 저장됩니다.

또는 파일을 이 영역으로 끌어다 놓으세요 · 지원 형식 : .xlsx
2. 업로드 미리보기
"이 기기에 저장"을 누르면 즉시 요약/보고/통계 페이지에 반영됩니다. 다른 컴퓨터·팀원과 공유하려면 아래 3번에서 JSON을 내려받아 저장소(GitHub)의 data/ 폴더에 커밋하세요.
3. 저장된 주차 데이터 관리
4. JSON 내보내기 (저장소 커밋용)

GitHub Pages는 정적 호스팅이라 서버에 자동 저장이 되지 않습니다. 아래 버튼으로 JSON을 내려받아 저장소에 커밋하면 모든 기기·팀원이 같은 데이터를 보게 됩니다 (관리 구조는 하단 안내 참고).

데이터 저장 구조 안내
커밋 데이터 (저장소 /data 폴더, 모든 사람 공유)
· data/index.json — 저장된 전체 주차 날짜 목록
· data/weekly/{기준일자}.json — 해당 주차의 시군구별/점검결과/업체별 원본 스냅샷 (통합본)

로컬 데이터 (이 브라우저에만 저장, localStorage)
· 방금 업로드했지만 아직 저장소에 커밋하지 않은 주차. 다른 기기에서는 보이지 않습니다.

시군구별 / 업체별 JSON은 모든 저장된 주차 스냅샷을 모아 지역·업체 단위 시계열로 재구성한 파생 자료입니다. 화면(요약/보고/통계)은 이 파일을 읽지 않고 weekly/*.json에서 매번 직접 계산하므로, 이 두 파일은 외부 공유·엑셀 분석용으로만 사용하면 됩니다.
